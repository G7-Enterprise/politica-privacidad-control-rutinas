## 2. Resumen Ejecutivo

Biblioteca Legal NICA es una aplicación móvil que proporciona acceso a la legislación nicaragüense de forma offline y online. Esta política de privacidad describe cómo recopilamos, utilizamos y protegemos su información personal.

## 3. Información que Recopilamos

### 3.1 Información NO Recopilada
- **No recopilamos datos personales** de los usuarios
- **No recopilamos información de contacto** (nombres, emails, teléfonos)
- **No recopilamos ubicación geográfica**
- **No recopilamos información de dispositivos** para identificación personal
- **No recopilamos datos de uso** para análisis de comportamiento

### 3.2 Información Técnica (Anónima)
- **Versión de la aplicación** (para actualizaciones)
- **Tipo de dispositivo** (para compatibilidad)
- **Versión de Android** (para soporte técnico)
- **Permisos concedidos** (para funcionalidad de la app)

## 4. Permisos de la Aplicación

### 4.1 Permisos Requeridos
- **INTERNET:** Para descargar actualizaciones de contenido legal
- **ACCESS_NETWORK_STATE:** Para verificar conectividad
- **WRITE_EXTERNAL_STORAGE:** Para guardar contenido offline
- **READ_EXTERNAL_STORAGE:** Para acceder a contenido guardado
- **WAKE_LOCK:** Para mantener la pantalla activa durante lectura
- **VIBRATE:** Para notificaciones de actualizaciones

### 4.2 Permisos Opcionales
- **RECORD_AUDIO:** Para funcionalidad de búsqueda por voz (si se implementa)
- **CAMERA:** Para escanear códigos QR de artículos legales (si se implementa)
- **LOCATION:** Para mostrar leyes específicas por región (si se implementa)

## 5. Uso de la Información

### 5.1 Propósito Principal
- **Acceso a legislación:** Proporcionar acceso a leyes nicaragüenses
- **Funcionalidad offline:** Permitir consulta sin conexión a internet
- **Actualizaciones:** Notificar sobre nuevas leyes o modificaciones
- **Soporte técnico:** Resolver problemas de la aplicación

### 5.2 Text-to-Speech (TTS)
- **Procesamiento local:** El TTS funciona completamente en el dispositivo
- **Sin envío de datos:** El texto no se envía a servidores externos
- **Voces locales:** Utiliza voces instaladas en el dispositivo

## 6. Almacenamiento de Datos

### 6.1 Datos Locales
- **Contenido legal:** Almacenado localmente en el dispositivo
- **Configuraciones:** Preferencias de usuario guardadas localmente
- **Cache:** Datos temporales para mejorar rendimiento

### 6.2 Datos en la Nube
- **No almacenamos datos personales** en servidores externos
- **Contenido legal:** Descargado desde fuentes oficiales (La Gaceta)
- **Actualizaciones:** Verificadas contra repositorios oficiales

## 7. Compartir Información

### 7.1 No Compartimos Datos Personales
- **No vendemos** información personal
- **No compartimos** con terceros
- **No utilizamos** para publicidad

### 7.2 Excepciones Legales
- **Orden judicial:** Solo si es requerido por ley
- **Protección de derechos:** Para proteger nuestros derechos legales
- **Seguridad:** Para prevenir fraudes o actividades ilegales

## 8. Seguridad de Datos

### 8.1 Medidas de Seguridad
- **Cifrado local:** Datos sensibles cifrados en el dispositivo
- **Comunicación segura:** HTTPS para todas las comunicaciones
- **Validación de integridad:** Verificación de contenido descargado
- **Actualizaciones de seguridad:** Parches regulares de seguridad

### 8.2 Protección de Contenido
- **Verificación de fuentes:** Contenido legal verificado contra fuentes oficiales
- **Integridad de datos:** Checksums para verificar integridad
- **Backup automático:** Respaldos de configuración local

## 9. Derechos del Usuario

### 9.1 Control de Datos
- **Acceso:** Puede ver toda la información almacenada localmente
- **Modificación:** Puede cambiar configuraciones en cualquier momento
- **Eliminación:** Puede desinstalar la aplicación para eliminar todos los datos
- **Exportación:** Puede exportar contenido legal para uso personal

### 9.2 Transparencia
- **Código abierto:** Partes del código disponibles en GitHub
- **Auditorías:** Revisiones regulares de seguridad
- **Reportes:** Informes de transparencia disponibles
